# Veilarbdemo

Eksempel repo med java oppsett som bygger og kjører på nais.

## Kjøre appen
```console
# bygge
mvn clean install 

# test
mvn test

# starte
# Kjør main-metoden i Main.java
# For lokal test kjøring kjør MainTest.java
```
